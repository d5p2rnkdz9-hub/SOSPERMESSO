  <action>
Find the existing "List inside cards" section in main.css (around line 228-237).

Replace or extend the existing `.card ul, .card ol` and `.card li` rules with triangle bullet styling:

```css
/* List inside cards - Triangle bullets */
.card ul {
  list-style: none;
  margin-top: var(--spacing-sm);
  padding-left: var(--spacing-lg);
}

.card ol {
  margin-top: var(--spacing-sm);
  padding-left: var(--spacing-lg);
}

.card li {
  margin-bottom: var(--spacing-xs);
  position: relative;
  padding-left: 0;
}

.card ul li::before {
  content: '\25B8';
  position: absolute;
  left: -1.25rem;
  color: var(--taxi-yellow-dark);
  font-size: 0.85rem;
  line-height: 1.5;
}

/* Nested lists in cards */
.card ul ul {
  margin-top: var(--spacing-xs);
  margin-bottom: var(--spacing-xs);
}

.card ul ul li::before {
  content: '\25B9';
  font-size: 0.75rem;
}
```

Key points:
- Use `\25B8` (BLACK RIGHT-POINTING SMALL TRIANGLE) for primary bullets
- Use `\25B9` (WHITE RIGHT-POINTING SMALL TRIANGLE) for nested lists
- Use existing `--taxi-yellow-dark` color for brand consistency
- Position with `position: absolute` and `left: -1.25rem` to align with text

Do NOT use `!important` - proper specificity is sufficient.
